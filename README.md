# GeekUp Ecommerce Backend

A scalable backend application built with NestJS.

## Getting Started

### Prerequisites
- Node.js
- Docker
- pnpm

### Running the Application

```bash
# Install dependencies
pnpm install

# Run the local Docker environment:
pnpm docker:up:local
```
